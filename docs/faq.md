# Frequently Asked Questions

Get quick answers to common questions about using DearDonor.

## General Questions

### What is DearDonor?
DearDonor is a comprehensive donation management platform designed specifically for nonprofits. It helps organizations track donations, manage donor relationships, and generate reports efficiently.

### How secure is my data?
We implement industry-standard security measures including:
- End-to-end encryption
- Regular security audits
- GDPR compliance
- Secure data backups
- Role-based access control

### Can I import existing donor data?
Yes! DearDonor supports importing donor data from:
- CSV files
- Excel spreadsheets
- Common CRM systems
- Other donation platforms

## Donation Management

### How do I record a new donation?
:::feature-card

1. Navigate to the "Donations" section
2. Click "New Donation"
3. Fill in the donor information
4. Enter donation details
5. Save the record

:::

### Can I process recurring donations?
Yes, DearDonor supports recurring donations with flexible scheduling options:
- Daily
- Weekly
- Monthly
- Quarterly
- Annually

### How do I handle refunds?
:::tip-card

To process a refund:
1. Locate the donation in the system
2. Click "Process Refund"
3. Enter refund amount and reason
4. Confirm the refund
5. The system will automatically update records

:::

## Donor Management

### How do I add a new donor?
To add a new donor:
1. Go to "Donors" section
2. Click "Add New Donor"
3. Fill in contact information
4. Set communication preferences
5. Save the profile

### Can I track donor engagement?
Yes! DearDonor provides multiple ways to track donor engagement:

- Custom engagement metrics

### How do I manage donor communications?
:::feature-card

DearDonor offers various communication tools:

- Automated thank-you notes
- Campaign tracking

:::

## Account Management

### How do I update my account settings?
:::warning-card

To update your account:
1. Go to "Settings"
2. Select the section to update
3. Make your changes
4. Save the updates
5. Verify the changes

:::

### What if I forget my password?
To reset your password:
1. Click "Forgot Password"
2. Enter your email
3. Check your inbox
4. Follow the reset link
5. Create a new password

### Can I have multiple users?
Yes! DearDonor supports multiple user roles:
- Administrators
- Staff members
- Volunteers
- Read-only users
- Custom roles
